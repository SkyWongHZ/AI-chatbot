AI-chatbot/
├── .github/                # GitHub Actions 自动部署配置 (可选)
├── .vscode/                # VS Code 编辑器配置 (推荐)
├── docs/                   # 项目文档 (API文档、设计图、需求说明)
├── packages/               # 核心代码目录
│   ├── client/             # 【前端】代码 (Vue/React)
│   │   ├── public/         # 静态资源 (favicon, index.html)
│   │   ├── src/
│   │   │   ├── assets/     # 图片、字体、全局样式
│   │   │   ├── components/ # 公共组件 (Button, ChatBubble)
│   │   │   ├── views/      # 页面视图 (Home, AdminPanel)
│   │   │   ├── hooks/      # 自定义 Hooks
│   │   │   ├── store/      # 状态管理 (Pinia/Redux)
│   │   │   ├── api/        # 前端 API 请求封装
│   │   │   ├── utils/      # 前端工具函数
│   │   │   ├── App.vue
│   │   │   └── main.ts
│   │   ├── .env            # 前端环境变量
│   │   └── package.json    # 前端依赖
│   │
│   └── server/             # 【后端】代码 (Node.js/Python)
│       ├── src/
│       │   ├── config/     # 配置文件 (数据库配置, AI Key)
│       │   ├── controllers/# 控制器 (处理请求逻辑)
│       │   ├── models/     # 数据库模型 (Schema)
│       │   ├── routes/     # 路由定义
│       │   ├── services/   # 业务逻辑层 (调用 AI 接口的核心逻辑在这里)
│       │   ├── middlewares/# 中间件 (鉴权, 日志)
│       │   ├── utils/      # 后端工具函数
│       │   └── app.ts      # 入口文件
│       ├── .env            # 后端环境变量 (敏感信息)
│       └── package.json    # 后端依赖
│
├── scripts/                # 自动化脚本 (如一键启动前后端)
├── .gitignore              # Git 忽略文件
├── README.md               # 项目说明书
└── package.json            # 根目录配置 (管理整个工作区)


- client目录下用vue3和vite,数据状态管理用pinia创建前端项目；
- server目录下用koa-generaotr直接生成一个可用的后端项目
